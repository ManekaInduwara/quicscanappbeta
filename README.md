# QuickScanner

QuickScanner is a Flutter-based mobile application designed for fast, simple, and secure event ticket scanning. It helps event organizers, merchants, and gate staff scan QR codes and verify tickets during event check-ins.

This app is built as part of the QuickPass ticketing ecosystem and is optimized for Android release builds.

---

## Overview

QuickScanner allows authorized event staff to validate tickets quickly using a mobile device. The app is useful for concerts, festivals, live shows, private events, and other ticketed experiences where QR-based ticket verification is required.

---

## Features

- QR code ticket scanning
- Fast ticket verification
- Simple and clean user interface
- Mobile-friendly scanning flow
- Secure ticket validation process
- Suitable for event entrance check-ins
- Android release build support
- Optimized APK and App Bundle build commands
- Supports production deployment workflow
